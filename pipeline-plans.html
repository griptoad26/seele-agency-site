⚡ Pipeline 3: Task Execution (Autonomous Runner)
trigger: cron */5 * * * *
01 · Poll
GET /api/tasks?status=pending
every 5 min, max 5 tasks per cycle
↓
02 · Decision Gate
Pending tasks exist?
↙ NO ─── ↘ YES
EXIT
Idle until next cycle
PROCESS
03 · Route
Match title/desc → agent
agent: pipeline-router.sh
build→build, test→qa, deploy→devops...
↓
04 · Mark In-Progress
PATCH /api/tasks/<id> {status:in_progress}
↓
05 · Execute
Spawn agent or run pipeline script
agent: <routed-agent-id>
timeout 60-90s based on pipeline
↓
06 · Decision Gate
Exit 0?
↙ NO ─── ↘ YES
RETRY
status=delegated, log error, retry next cycle
COMPLETE
07 · Close
PATCH /api/tasks/<id> {status:completed}
append to autonomous-runner.log
↓
✓ Task closed, next task in queue
